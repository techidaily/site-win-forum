## So what does a BIOS include?

 The BIOS includes instructions on how to load basic computer hardware. BIOS tests your system to verify the computer meets requirements to boot up properly. If the computer does not pass the test, you hear a combination of beeps indicating what is malfunctioning in the computer.

## What are the four main functions of a PC BIOS?

* **POST** ‚Äì BIOS performs a test on your computer hardware and make sure no errors exist before loading the operating system.
* **Bootstrap Loader** ‚Äì This function locates the operating system. If a capable operating system is located, the BIOS will pass control to it.
* **BIOS drivers** ‚Äì Low-level drivers that give the computer basic operational control over your computer‚Äôs hardware.
* **BIOS setup or CMOS setup** ‚Äì Configuration program that allows you to configure hardware settings including system settings, such as date, time, and computer passwords.

## How does BIOS work?

 Most computers come with included BIOS in them as firmware on the chip of your motherboard. BIOS is a program that is accessible by the microprocessor on an erasable programmable read-only memory (EPROM) chip. When you turn on your PC the microprocessor passes control to the BIOS.

 When you first start your computer BIOS checks if your system is operational. If everything is working on your system BIOS load the OS into the computer‚Äôs RAM (random access memory) from your hard disk.

1. Reset or power off the computer.
2. When the computer turns back on, look for a message that says ‚Äúentering setup‚Äù or something similar. ## BIOS security

 Usually not a lot of users talk about BIOS security, but it should not be overlooked. There is a case study by the Security Group ‚Äì Cylance that showed how modern BIOS can have flaw in its security and ransomware programs can access the motherboard‚Äôs UEFI and exploit the other PC BIOS vulnerabilities.

 Another unique exploit involving the manipulation of BIOS was Plundervolt. Plundervolt could be used to mess with a computer‚Äôs power supply at the time data was being written to memory, causing errors that lead to security gaps. Intel released a BIOS patch to defend against it.

## Can a BIOS chip be upgraded or updated?

 Adding additional memory to a BIOS chip, as an upgrade, can only be done by replacing the existing BIOS chip with a new, more advanced BIOS chip.

 The data on a BIOS chip can be updated if it‚Äôs a flash BIOS. Using specially designed software, the BIOS can be updated to fix problems or add new features for the motherboard.
